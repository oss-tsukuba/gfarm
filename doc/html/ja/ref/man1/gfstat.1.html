Name

gfstat — Gfarm ファイルまたはディレクトリの情報を表示する

Synopsis

gfstat [options] path...

DESCRIPTION

指定したファイルあるいはディレクトリの情報を表示します．

OPTIONS

-M

メタデータ・サーバの情報も表示します．

-c

ファイル複製の数のみを表示します．

-l

シンボリックリンクを辿らず．リンク自身の情報を表示します．

-r

メタデータではなく実際のファイルデータのファイルサイズを表示します．書き込み中のファイルのファイルサイズを表示することができます．

-h host

-r オプションとともに用いられ、ファイルが格納されているホストを指定します．

-?

引数オプションを表示します．
