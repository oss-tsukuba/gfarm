Name

gfs_acl_set_tag_type — set the tag type of a Gfarm ACL entry

Synopsis

#include <gfarm/gfarm.h>
gfarm_error_t gfs_acl_set_tag_type( entry_d,  
  tag_type);  
gfarm_acl_entry_t entry_d;
gfarm_acl_tag_t tag_type;
 

INPUT ARGUMENTS

entry_d

An ACL entry.

tag_type

An ACL tag type.

RETURN VALUES

GFARM_ERR_NO_ERROR

The function terminated successfully.

Others

An error except the above occurred. The reason is shown by gfarm_error_string(3).
