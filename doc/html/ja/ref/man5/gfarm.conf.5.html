gfarm.conf

Name

gfarm.conf -- Gfarm 肋年ファイル

DESCRIPTION

gfarm.conf ファイルは、gfarm プログラムの肋年ファイルです。 サ〖バ〖ˇプロセスである gfmd と gfsd は、デフォルトでは、 この %%SYSCONFDIR%%/gfarm.conf を肋年ファイルとして徊救します。 なお、肋年柒推が粕み哈まれるのは弹瓢箕だけなので、 肋年柒推を恃构した眷圭、サ〖バ〖を浩弹瓢する涩妥があります。

アプリケ〖ションプログラムは、%%SYSCONFDIR%%/gfarm.conf と、ユ〖ザ〖の ホ〖ムディレクトリにある .gfarmrc ファイルを、肋年ファイル として徊救します。.gfarmrc ファイルは、%%SYSCONFDIR%%/gfarm.conf と 票办の妨及で淡揭します。.gfarmrc と %%SYSCONFDIR%%/gfarm.conf では、 .gfarmrc の数が庭黎されます。

gfarm.conf は、办乖に办矢の妨及で肋年します。 ただし、乖琐に矢机 ``\'' を淡揭することによって、乖を费鲁させることが できます。また、矢机 ``#'' から乖琐までは、コメントとして胺われ、痰浑 されます。

ホスト回年

苞眶として、ホスト回年 を淡揭できる矢がいくつかありますが、 この眷圭、ホスト回年には、布淡のいずれかを淡揭できます。

III.JJJ.KKK.LLL

"." で惰磊られた 0×255 までの眶机 4 つで、IP アドレスを回年します。

III.JJJ.KKK.LLL/MM

IP アドレスと、"/" で惰磊られた 0×31 までの眶机で、ネットワ〖クを 回年します。"/" 笆惯の眶机は netmask のビット墓です。

domain.name

ホスト叹をドメイン叹で回年します。

.domain.name

ドメイン叹の黎片が "." ではじまる眷圭、このドメイン叹に疥掳する链ての ホストを罢蹋します。

*

"*" と淡揭されている眷圭、链てのホストを罢蹋します。

LISTENER

sockopt 矢の眷圭、链て络矢机で "LISTENER" と淡揭すると、されている眷圭、链てのホストを罢蹋します。

GSI千沮において、gfsdが脱いる沮汤今の硷梧を回年します。 sharedsecret 网脱箕には、この回年は帽に痰浑されます。

サ〖バ娄でこの回年を臼维した眷圭、サ〖バを root 涪嘎で弹瓢した眷圭は ホスト沮汤今を脱います。またサ〖バを办忍ユ〖ザ涪嘎で弹瓢した眷圭は ユ〖ザ沮汤今を脱います。このため、サ〖ビス沮汤今を网脱する眷圭を 近いて、サ〖バ娄ではこの回年を乖う涩妥は奶撅ないでしょう。

クライアント娄でこの回年を臼维した眷圭、サ〖バがホスト沮汤今を网脱している と簿年してサ〖バ千沮を乖います。このためサ〖バをクライアントと票じユ〖ザ 涪嘎で弹瓢している眷圭、クライアント娄では、布淡の办乖を肋年して おく涩妥があります。

回年材墙な cred_type としては、 ≈self∽、 ≈host∽、≈user∽、 ≈mechanism-specific∽があり、 spool_server_cred_service および spool_server_cred_name と寥み圭わせて 布淡のように脱います。

ホスト沮汤今あるいはサ〖ビス沮汤今を脱いることを绩します。 サ〖ビス沮汤今を脱いる眷圭、そのサ〖ビスの叹涟を spool_server_cred_service で回年します。 サ〖ビス叹に≈host∽を回年した眷圭、 ファイル ``/etc/grid-security/hostcert.pem'' に あるホスト沮汤今を脱います。 それ笆嘲のサ〖ビス叹を回年した眷圭、 ファイル ``/etc/grid-security/サ〖ビス/サ〖ビスcert.pem'' にあるサ〖ビス沮汤今を脱います。 サ〖ビス叹を臼维した眷圭、≈host∽が回年されたとみなします。 ホスト沮汤今およびサ〖ビス沮汤今の眷圭、沮汤今の Common Name 腕のみを 孺秤します。Common Name は、≈CN=サ〖ビス叹/ホスト叹∽の妨及である涩妥が あります。また、ホスト叹は、gfhost コマンドで回年した 赖及叹と赖澄に办米している涩妥があります。ホスト侍叹は钓されません。

これは、GSSAPI (RFC2743/RFC2744) の GSS_C_NT_HOSTBASED_SERVICE に 滦炳する怠墙です。

user

ユ〖ザ沮汤今を脱いることを绩します。 ユ〖ザのアカウント叹を spool_server_cred_name で回年します。 アカウント叹を臼维した眷圭には、そのコマンドを弹瓢したユ〖ザ叹が蝗われます。 この回年を网脱する狠には、spool_server_cred_service の 回年を乖ってはいけません。

附箕爬では、この怠墙の悸刘に、いくつかの悸刘惧の扩嘎祸灌があります。 まず、ユ〖ザ叹はロ〖カルホストのアカウント叹として豺坚されますが、 これはバグであり、gfarm のグロ〖バルユ〖ザ叹に恃构される徒年です。 ユ〖ザ叹と沮汤今の Distinguished Name の滦炳をとるため、 ``/etc/grid-security/grid-mapfile'' を脱います。 このため、このファイルが赂哼しなかったり、あるいはこのファイルに判峡 されてないユ〖ザは、この怠墙を脱いることができません。 さらに、このファイル面で、票办のユ〖ザアカウントに滦して2つ笆惧の Distinguished Name が肋年されていた眷圭、2つ叹笆惯の肋年は痰浑されて しまいます。

これは、GSSAPI (RFC2743/RFC2744) の GSS_C_NT_USER_NAME に滦炳する怠墙です。

これは、GSSAPI (RFC2743/RFC2744) で Name Type として GSS_C_NO_OID を 回年した眷圭に滦炳する怠墙です。

auth 铜跟痰跟 千沮数恕 ホスト回年

この矢は、妈3苞眶で回年したホストとの奶慨に脱いる千沮数恕を肋年します。

妈1苞眶の铜跟痰跟婶には、enable ないし disable キ〖ワ〖ドを回年します。 妈2苞眶の千沮数恕婶には、gsi、 gsi_auth ないし sharedsecret キ〖ワ〖ドを回年します。 妈3苞眶には、ホスト回年を淡揭します。

この矢は剩眶回年材墙です。称千沮数恕ごとに、黎片から界にホスト回年に 努圭するかどうか拇べ、铜跟であるとの回年に努圭した眷圭、その千沮数恕が 网脱铬输になります。铜跟であるとの回年に努圭しない眷圭や、あるいは 铜跟であるとの回年に努圭するよりも涟に痰跟であるとの回年に努圭した 眷圭、その千沮数恕は、铬输になりません。

この回年は、サ〖バ〖娄とクライアント娄の尉数で豺坚され、 列数ともで铜跟になっている千沮数恕のみが脱いられます。

千沮数恕が佰なるものに簇しては、回年の界进は罢蹋がありません。 剩眶の千沮数恕が铬输となった眷圭、sharedsecret、 gsi_auth、gsi 千沮の界进で活みます。

Gfarm のコンパイル箕に globus とのリンクを回年しなかった眷圭、 GSI は网脱できません。この眷圭、gsi および gsi_auth 千沮の回年は帽に痰浑されます。

この肋年は涩寇です。

毋:

auth disable sharedsecret 192.168.0.100
auth disable sharedsecret 192.168.0.101
auth enable sharedsecret 192.168.0.0/24
auth enable gsi_auth 10.0.0.0/8
auth enable gsi *

この毋では、ホスト 192.168.0.100 と 192.168.0.101 の 2 つのホストを 近いた 192.168.0.0/24 ネットワ〖クに掳すホストには sharedsecret と gsi の尉数を、 10.0.0.0/8 ネットワ〖クに掳すホストには gsi_auth および gsi の尉数を、 それ笆嘲の链てのホストには gsi 千沮のみを活みます。 すなわち、ホスト 192.168.0.100 と 192.168.0.101 の 2 つのホストは、 gsi千沮のみを活みることになります。

netparam パラメ〖タ=猛 [ホスト回年]

ネットワ〖ク奶慨に脱いる称硷パラメ〖タを回年します。 パラメ〖タには布淡のものがあります。

parallel_streams。このパラメ〖タは、奶慨に网脱する TCPのストリ〖ム眶を回年します。臼维箕は1です。 このパラメ〖タは、TCP の奶慨ウィンドウが浇尸なサイズ まで倡かないような斌数との奶慨の狠に、铜跟なことがあります。 附哼、このパラメ〖タはファイルの剩澜怠墙でのみ铜跟です。 附哼の悸刘では、gfrep(1) コマンドを弹瓢したホストのユ〖ザ〖肋年ファイルの 柒推が脱いられます。ホスト回年と孺秤されるのは、剩澜黎のホストではなく、 剩澜傅のホストです。

stripe_unit_size。このパラメ〖タは、奶慨に剩眶の TCP ストリ〖ム を网脱する眷圭に、どのように啪流柒推を尸充するかを回年します。 もしこの回年がない眷圭、啪流柒推を帽に parallel_streams 改に 尸充して流慨します。 これに滦し、stripe_unit_size を回年した眷圭は、このパラメ〖タ で回年したバイト眶を帽疤としてストライピングして流慨します。 奶撅、帽なる尸充よりもストライピングの数が拉墙弄に铜网です。

このパラメ〖タは、惧淡の parallel_streams として 1 より络きな猛 を回年した奶慨のみで铜跟です。 附哼、このパラメ〖タはファイルの剩澜怠墙でのみ铜跟です。 附哼の悸刘では、gfrep(1) コマンドを弹瓢したホストのユ〖ザ〖肋年ファイルの 柒推が脱いられます。ホスト回年と孺秤されるのは、剩澜黎のホストではなく、 剩澜傅のホストです。

rate_limit。コネクション1塑あたりの奶慨バンド升を扩嘎する 悸赋弄な怠墙です。回年する猛の帽疤は bps (bit per second) です。 附哼、このパラメ〖タはファイルの剩澜怠墙でのみ铜跟です。 附哼の悸刘では、剩澜傅のホスト惧の gfsd の肋年ファイルが脱いられます。 ホスト回年と孺秤されるのは、剩澜黎のホストです。

sockopt オプション[=猛] [LISTENER | ホスト回年]

奶慨 socket(2) に滦して、setsockopt(2) システムコ〖ルを蝗って ソケットˇオプションを回年します。

妈2苞眶に链て络矢机で LISTENER を回年した眷圭、 サ〖バ〖娄 (缅钙娄) のソケットに回年するオプションになります。 この眷圭、奶慨する陵缄に簇犯なく、 办围にソケットˇオプションが肋年されます。
