gfarm_stringlist_cat

Name

gfarm_stringlist_cat -- stringlist に文字列配列を追加する

Synopsis

#include <gfarm/gfarm.h>

char *gfarm_stringlist_cat(gfarm_stringlist * listp, char ** v);

DESCRIPTION

listp の表す文字列配列の末尾に、NULL で終了する文字列配列 v を追加します。

RETURN VALUES

NULL

正常終了を示します。

GFARM_ERR_NO_MEMORY

メモリ不足のため処理を中止しました。

その他

上記以外のエラーが生じたことを示します。
