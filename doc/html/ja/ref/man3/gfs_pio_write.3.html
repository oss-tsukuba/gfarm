gfs_pio_write

Name

gfs_pio_write -- ファイルの書き込み

Synopsis

#include <gfarm/gfarm.h>

char *gfs_pio_write(GFS_File gf, void * buffer, int size, int * np);

DESCRIPTION

bufferから size バイトだけ、ファイル gf へ書き込みます。 実際に書き込まれたバイト数は np の指す先に返します。

RETURN VALUES

NULL

正常終了を示します。

GFARM_ERR_NO_MEMORY

メモリ不足のため処理を中止しました。

その他

上記以外のエラーが生じたことを示します。

gfs_pio_create(3), gfs_pio_flush(3), gfs_pio_open(3)
