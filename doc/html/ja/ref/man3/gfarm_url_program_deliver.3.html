gfarm_url_program_deliver

Name

gfarm_url_program_deliver -- プログラムを指定したホストに配布する

Synopsis

#include <gfarm/gfarm.h>

char *gfarm_url_program_deliver(char *gfarm_url, int nhosts, char **hosts, char ***delivered_paths);

DESCRIPTION

gfarm_urlで指定したプログラムを、 nhosts 個の要素を持つ文字列配列 hosts で指定したホストに配布します。 配布された各ホスト上でのプログラムの絶対パスを、 nhosts 個の要素を持つ文字列配列に格納し、その配列のアドレスを delivered_paths が指す先に返します。 delivered_paths

に得た文字列配列は、不要になったら gfarm_strings_free_deeply(3) で解放する必要があります。

RETURN VALUES

NULL

正常終了を示します。

GFARM_ERR_NO_MEMORY

メモリ不足のため処理を中止しました。

その他

上記以外のエラーが生じたことを示します。
