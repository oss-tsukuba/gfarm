gfwhere

Name

gfwhere -- print a replica catalog of each Gfarm fragment

Synopsis

gfwhere [-I fragment-index] [-s] gfarm-URL...

DESCRIPTION

gfwhere prints a replica catalog of a Gfarm file specified by gfarm-URL. When the Gfarm file is a regular fragmented file, gfwhere prints in the following format.

fragment index: filesystem node ...

When the Gfarm file is an executable file, gfwhere prints in the following format.

architecture: filesystem node ...

OPTIONS

-I fragment-index

Specifies the fragment index to be displayed. If the gfarm-URL is an executable file, an architecture name, for example, sparc-sun-solaris8, can be used to specify the executable architecture to be displayed.

-s

Displays the size of each file fragment.

-?

Displays a list of command options.
