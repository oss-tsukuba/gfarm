gfsched

Name

gfsched -- create a host file using file-affinity scheduling

Synopsis

gfsched [-I fragment-index] [-p gfarm-program-URL] gfarm-URL [output-hostfile]

gfsched -N number [-p gfarm-program-URL] [host...]

DESCRIPTION

gfsched creates a host file using the file-affinity scheduing of gfarm-URL. In other words, gfsched chooses and displays hostnames of the same number with the fragments of the gfarm-URL, with giving priority to the hosts which has the fragment as far as possible. When output-hostfile is omitted, gfsched displays to the standard output.

If -N is specified instead of gfarm-URL, gfsched chooses and displays specified number of hosts from the hosts in the arguments. If no host is specified in the argument, gfsched chooses hosts from all registered hosts.

OPTIONS

-I fragment-index

specifies the fragment index to be scheduled.

-p program-URL

creates a host file with a program name which will be used to process the gfarm-URL. In other words, only hosts which can run the program-URL will be selected, if this option is specified.

-?

displays a list of command options.
