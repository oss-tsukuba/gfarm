gfarm_stringlist_free

Name

gfarm_stringlist_free, gfarm_stringlist_free_deeply -- stringlist を解放する

Synopsis

#include <gfarm/gfarm.h>

void gfarm_stringlist_free(gfarm_stringlist * listp);

void gfarm_stringlist_free_deeply(gfarm_stringlist * listp);

DESCRIPTION

動的に確保したメモリを解放します。

関数 gfarm_stringlist_free() は、変数 listp が表す文字列配列の要素である各文字列については解放しません。

これに対し、関数 gfarm_stringlist_free_deeply() は、文字列配列の要素で ある各文字列も解放します。
