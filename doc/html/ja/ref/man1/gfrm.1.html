gfrm

Name

gfrm -- Gfarm ファイル、ディレクトリ、ファイル複製を削除する。

Synopsis

gfrm [-frR] [-h ファイルシステムノード] gfarm-URL...

gfrm [-f] -h ファイルシステムノード -I フラグメント番号 gfarm-URL...

DESCRIPTION

Gfarm-URL に指定されたファイルを削除します。 デフォルトではディレクトリの削除は行いませんが、 -r あるいは -R オプションが指定された場合、 指定されたディレクトリ以下のファイルおよびディレクトリが再帰的に消去されます。 なお、. あるいは .. が指定された場合はエラーになります。 -h オプションでファイルシステムノードを指定することにより、 指定されたノードに格納されているファイル複製を削除することができます。 ただし、全てのファイル複製を削除することはできません。

また、 -h オプションと -I オプションの両方を指定することにより、 特定のファイル断片の複製を削除することができます。 ただし、全てのファイル複製を削除することはできません。

OPTIONS

-f

ファイル複製削除の場合に、 全てのファイル複製を削除することになってもファイル複製を削除します。 この時、ファイル自体が削除されることになります。 このオプションは -h オプションとともに指定された場合に有効です。

-r, -R

引数がディレクトリだった場合、そのディレクトリ以下を 全て、再帰的に削除します。

-h ファイルシステムノード

複製を削除するファイルシステムノード名を指定します。 複数のファイルシステムノードか ら同じファイル断片の複製を一度に削除するためには，このオ プションを複数回指定します．

-I フラグメント番号

複製を削除するフラグメント番号を指定します。 実行形式のファイルの場合には、このオプションで実行アーキ テクチャ(例 「sparc-sun-solaris8」) を指定することによっ て、指定したアーキテクチャの複製を削除できます。

-?

引数オプションを表示します。
