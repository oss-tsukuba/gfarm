gfs_pio_flush

Name

gfs_pio_flush -- 書き込みをフラッシュする。

Synopsis

#include <gfarm/gfarm.h>

char *gfs_pio_flush(GFS_File gf);

DESCRIPTION

書き込み用にオープンしているファイル gf の、書き込みバッファに溜まっているデータを実際にディスクに書き込みます。

RETURN VALUES

NULL

正常終了を示します。

GFARM_ERR_NO_MEMORY

メモリ不足のため処理を中止しました。

その他

上記以外のエラーが生じたことを示します。
